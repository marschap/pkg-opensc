ASEPCOS

 Athena smartcards and tokens are supported as for OpenSC-0.11.6.

Fully PKCS#15 emulation is supported.

Athena fully cooperate with the OpenSC project, and provide any required information.

If you interested in Athena's tokens (ASEKey), make sure you order the CCID complaint version.

Another issue you may encouter is failure initializing the PKCS#15 structure. This may be due to pre-formated smartcard using Athena proprietary provider. If you have this issue, ask for help in OpenSC mailing list.
