Mac OS X CSP

Apple has also implemented a powerful schema for cryptography.

Unfortunatly currently there is no implementation for OpenSC to work with those mechanism.

The Apple Developer Library includes documents on these issues, for example Apple Cryptographic Service Provider Functional Specification.

Apple has a mailing list about the topic. An Archive of that list is also available.

Also take a look at the Common Data Security Architecture documentation (CDSA) published by the Open Group.

UPDATE: we're (somewhat) working on this: see http://www.opensc-project.org/opensc/wiki/MacOsXTokend
