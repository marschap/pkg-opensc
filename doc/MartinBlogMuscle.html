What i have

  • USB e-gate token for Cyberflex cards

What i run on

  • WindowsXP SP2 (x86, free visual c toolkit, .net sdk, mingw setup)
  • GNU/Linux (x86, kernel 2.6, mostly Debian/unstable)
  • OS X 10.4 (Xcode et al, fink)

How to access the e-gate USB token

  • Windows: http://www.reflexreaders.com/Support/software.html (works well)
  • Linux:
    • OpenCT provides support directly to OpenSC, also a IfdHandler? for pcsc-lite (openct works, pcsc-lite integration needs to be checked, problems with the device file option in reader.conf)
    • ifd-egate available here: http://secure.netroedge.com/~phil/egate/ with minor modifications compiles cleanly and works well on linux)
  • OSX: openct and-or ifd-egate and-or normal supported reader?

How to load the applet to the card

  • http://www.identityalliance.com/CardManagerClient-1.0.2.tar.gz to use a card management server by identityalliance
    • compiled smoothly on linux and osx
    • Did do something as well on Linux (Had to download the new certificate before). Procedure: select card, select applet, go! Very nice.
    • use http://archives.neohapsis.com/archives/dev/muscle/2005-q2/0241.html as rootcert.pem
